# 昆仑谣 v0.1.0 — 首版发布

> 一段从昆仑山出发的中华文化回望之旅。8 个节点、可顺听亦可质疑，用 AI 即时生成的对白陪你走完一遍。

## 这是什么

**昆仑谣**是一款 Windows 桌面互动叙事应用。你将在虚构角色"昆仑子"的引导下，从昆仑山的天柱想象出发，沿着创世神话 → 三皇五帝 → 礼乐与诸子 → 长安丝路 → 宋元明清 → 近代文物南迁 → 当代国风复兴一路走过来。每个节点都可以选择"顺听"或"质疑"，态度会影响故事的最终升华。

**它不是什么**：不是教科书，不是无脑选择题，不是预先写死的剧本。每一段对白都由本地或远端的大语言模型实时生成，结合 RAG 检索的文献片段确保不"瞎编"。

## 主要功能

### 走通 8 节点主线

- 序章 + 8 节点 transition + 升华结局，约 30–60 分钟一周目。
- 每个节点带 4 条顺听 + 4 条质疑文案，紧扣该节点的专属意象（昆仑→天柱/西王母；近代→西南联大/故宫南迁；当代→文化自觉/B 站国风…）。
- 节点切换时背景音乐平滑过渡。

### 双档位 AI 后端

- **本地档**：开箱使用 Qwen 2.5 3B（约 2 GB），可手动升级到 7B Pro 或降级到 1.5B Lite。GPU 加速走 Vulkan，CPU-only 也能跑。
- **远端档**：兼容 OpenAI Chat Completions 协议，可接 DeepSeek / 月之暗面 / 智谱 GLM / 自建网关。
- 设置面板带"测试连接"按钮，三秒判断 API Key、Base URL、模型名是否能跑通。

### 知识检索（RAG）

- 内置 52 条中华文化知识条目（昆仑神话、三皇五帝、仰韶考古、诸子百家、长安丝路、宋词元曲、心学、西南联大、当代国风…），每个节点拉到的内容只在该节点允许的话题范围内，避免剧透或跑题。

### 进度与隐私

- 节点粒度续读：关掉重开自动接上次进度。
- 远端 API Key 通过 Windows DPAPI（macOS Keychain / Linux libsecret）加密落盘，磁盘上看不到明文。
- 渲染进程沙箱化，所有原生能力走严格的 IPC 契约。

## 安装

1. 从下方 Assets 下载 `昆仑谣-0.1.0-Setup.exe`（约 111 MB）。
2. 双击运行，可选择安装路径，会在桌面与开始菜单创建快捷方式。
3. 首次启动后，进入设置面板下载本地模型（约 2 GB），或填入你已有的兼容 API。

> Windows 11 / Windows 10（64 位）。暂未提供 macOS 与 Linux 安装包。

## 已知限制

- 部分节点仍使用占位背景图，正式美术资产将在后续版本补齐。
- 1.5B 兜底档在长上下文下偶尔出现重复或截断，建议默认使用 3B 档。
- 远端 API 走非流式时若网络抖动可能超时，可在设置切换到本地档兜底。
