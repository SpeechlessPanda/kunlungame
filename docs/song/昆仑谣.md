# 昆仑谣

迷失了昆仑之巅
荒弃了大道之源
漫灭了三皇神迹
远去了五帝黄炎
烟消了一卦开天
云散了琴笙心传
空一个三坟五典
枉一番上古三玄
釜山合符在绝地天通还
打将封神祀制礼作乐篇
礼崩乐坏散于野
诸子百家出世间
三千年文脉传华夏
周秦汉唐气冲天
能不忆长安宋元明清纳百川
胡海番洋乱了弦
东邪西毒剑走偏
古圣绝学归隐潜
阴生阳克运多舛
百年淬火起涅槃
风物长宜放眼观
归来还是昆仑山
