# 背景素材需求包（8 节点）

本目录用于统一管理主线 8 个节点的背景图需求。

- 每个节点一个独立子目录。
- 每个子目录包含 requirements.md，明确构图、色彩、文化符号、运动转场建议。
- 素材来源优先级：
  - 自生成（推荐）
  - 可商用免费素材（需二次确认许可）

参考站点（用于找灵感与素材，不直接复制原图）：

- https://unsplash.com/s/photos/chinese-landscape
- https://en.wikipedia.org/wiki/Visual_novel
- https://vndb.org/

统一技术规格：

- 比例：16:9
- 分辨率：至少 2560x1440（推荐 3840x2160）
- 可读性：底部 40% 区域需预留对话框可读空间（低对比噪点、少高频纹理）
- 交付格式：webp（主）+ png（备份）
